# yourpetandyou
This is a fullstack web app (MERN Stack) meant to keep tracks of all souvenirs between a pet and their master.

Masters will be able to post, retrieve, update and delete souvenir posts as they wish to.

## Tools
- Frontend
    - JavaScript
    - React 
    - Tailwindcss
- Backend
    - NodeJs
- Database
    - Mongo DB
- Version control
    - Git
